Laravel Artisan 命令行操作的介绍

作者:yzc    发布于: 2017-08-05

Laravel Artisan 命令

0x01 Artisan 介绍

Artisan 是 Laravel 提供的 CLI(命令行接口)，它提供了非常多使用的命令来帮助我们开发 Laravel 应用

0x02 Artisan 命令介绍

1.生成App Key

php artisan key:generate

2.生成控制器

php artisan make:controller

3.生成模型

php artisan make:model

4.生成授权策略

php artisan make:policy

5.生成seeder文件

php artisan make:seeder

6.执行迁移

php artisan migrate

7.回滚迁移

php artisan migrate:rollback

8.重置数据库

php artisan migrate:refresh

9.填充数据库

php artisan db:seed

10.进入tinker环境

php artisan tinker

11.查看路由列表

php artisan route:list

12.使用 help 来查看各个 Artisan 命令的帮助界面

php artisan help xxx

注：由于还在学习，只是总结了一些基础的命令，在今后的学习中要是再遇到相关的命令还会继续的学习
